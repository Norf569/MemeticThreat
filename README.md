# MemeticThreat

Данные проект представляет собой облачное хранилище данных. Его целью является обучение работы с ASP.NET Core методом learning-by-doing.

Сервер реализован на ASP.NET Core. Передача файлов по HTTP. Используются JWT и EF. Архитектура - монолитная, так как проект малого объёма.

Клиентская часть реализована на WPF. Архитектура - MVVC.
